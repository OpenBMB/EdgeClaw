# Chinese PII Router for EdgeClaw

专门检测中文场景下的个人隐私信息，支持身份证号、手机号、银行卡号、邮箱、地址等敏感信息的识别和脱敏。

## 功能特性

- 🔍 **身份证号检测**：18位身份证号，含校验位验证
- 📱 **手机号检测**：11位中国手机号
- 💳 **银行卡号检测**：16-19位银行卡号，支持Luhn算法验证
- 📧 **邮箱检测**：标准邮箱格式
- 🏠 **地址检测**：中文地址关键词匹配
- 🛡️ **智能脱敏**：根据风险等级自动脱敏处理

## 风险分级

| PII 类型 | 风险等级 | 处理方式 |
|---------|---------|---------|
| 身份证号 | S3 (高风险) | 本地处理，不上传云端 |
| 银行卡号 | S3 (高风险) | 本地处理，不上传云端 |
| 手机号 | S2 (中风险) | 脱敏后上传 |
| 邮箱 | S2 (中风险) | 脱敏后上传 |
| 地址 | S2 (中风险) | 脱敏后上传 |

## 脱敏规则

- **身份证号**：`110101********1234` (保留前6位和后4位)
- **手机号**：`138****8000` (保留前3位和后4位)
- **银行卡号**：`6222************0123` (保留前4位和后4位)
- **邮箱**：`t***@example.com` (保留首字母和域名)
- **地址**：`北京市朝阳区****` (保留省市区，脱敏详细地址)

## 配置示例

```json
{
  "privacy": {
    "routers": {
      "chinese-pii": {
        "enabled": true,
        "weight": 45,
        "type": "custom",
        "module": "./src/routers/chinese-pii.js",
        "options": {
          "detectIdCard": true,
          "detectPhone": true,
          "detectBankCard": true,
          "detectEmail": true,
          "detectAddress": true
        }
      }
    },
    "pipeline": {
      "onUserMessage": ["privacy", "chinese-pii", "token-saver"]
    }
  }
}
```

## 技术实现

### 检测引擎

1. **规则匹配**：正则表达式快速匹配
2. **校验验证**：
   - 身份证号：校验位算法验证
   - 银行卡号：Luhn算法验证
3. **语义分析**：地址关键词匹配

### 性能优化

- 检测延迟：~0ms（纯正则匹配）
- 支持批量检测多个 PII
- 按长度排序避免短匹配干扰
