Workflow

How planning works

The planner keeps the interaction in the dashboard while still reusing the plugin's task creation flow.

  1. Describe the goal you want the background worker to own.
  2. Answer the clarifying questions shown in the planner view.
  3. Review the created task in the Tasks page once it is queued.

Reminder

Planner-created tasks still use the plugin's default loop and cost budgets unless you later edit the plugin settings.
